Projet Personnel : ERP de Gestion des Stocks

2024

Réalisé pendant les heures libres

Description du projet :

Développement d'un ERP de gestion des stocks en C# avec Windows Forms et SQL Server, visant à gérer diverses fonctions de gestion de stock, de suivi et d’organisation.

Structure de la base de données :
  • Catalogue des produits
  • Gestion des produits
  • Suivi des commandes
  • Détail des commandes
  • Gestion des utilisateurs
  • Gestion des clients
Objectifs :

Mettre en pratique et renforcer les compétences en conception et développement d'applications, tout en résolvant des problématiques réelles et en acquérant une expertise approfondie dans la création d'ERP.

Technologies et compétences pratiquées :

C#, Windows Forms, SQL Server, Entity Framework (ORM)

Projet Personnel : Application de Vente de Voitures

11/2024 | Développement en cours

Réalisé en utilisant ASP.NET Core MVC et Entity Framework

Description du projet :

Développement d'une application de gestion de vente de voitures permettant de gérer les clients, les réservations, les rendez-vous, les ventes, et les catégories de véhicules. Utilisation de .NET Core avec Entity Framework pour la création des opérations CRUD (Create, Read, Update, Delete).

Structure de la base de données :
  • Catégorie
  • Client
  • Historique
  • Rendez-vous
  • Réservations
  • Utilisateurs
  • Véhicules
  • Ventes
Technologies et fonctionnalités :

Utilisation de ADO.NET et Entity Framework pour les migrations et la gestion des relations entre les tables avec ICollection dans les modèles. Mise en place d’un système d’authentification via .NET Identity pour la gestion des utilisateurs et des rôles.

Objectifs :

Créer une solution complète de gestion de vente de voitures, intégrant des fonctionnalités avancées de gestion de clients et de rendez-vous, tout en améliorant mes compétences en ASP.NET Core et Entity Framework.
